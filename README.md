Virtual Assistant (Python) 🤖💻

A fully-featured Virtual Assistant built with Python and Tkinter, capable of voice and text interaction, managing tasks, opening applications/websites, playing music, and more. This project demonstrates the integration of speech recognition, text-to-speech, GUI programming, and automation.

Features
🗣 Voice & Text Interaction

Accepts voice commands using speech_recognition.

Accepts text input through a GUI entry box.

Responds with text-to-speech using pyttsx3.

Handles greetings, jokes, and conversational responses.

💻 Program & Website Launcher

Launches popular applications like Notepad, Chrome, Spotify, VLC, Telegram.

Opens websites like YouTube, Google, ChatGPT, Facebook, Netflix, and more.

Supports Google search queries via voice/text command.

🎵 Music & Audio Controls

Plays music files stored locally.

Background music support using pygame.

Volume control and stop/play functionality.

🗓 Task Scheduler & Reminders

Add, remove, and list scheduled tasks.

Automatically reminds user at scheduled times.

⏱ PC Shutdown Control

Schedule a PC shutdown or cancel shutdown from GUI.

🤹‍♂️ Fun & Personality

Can tell and learn new jokes.

Has a personality: friendly, sarcastic, or emotional.

Dynamic greetings based on time of day.

📝 Logging & Memory

Logs all user commands in VA_log.txt.

Remembers tasks, jokes, favorite songs, websites, and last command in va_memory.json.

Built With

Python 3

Tkinter – for GUI

pyttsx3 – Text-to-Speech

speech_recognition – Voice commands

pygame – Music playback

threading – Background tasks
